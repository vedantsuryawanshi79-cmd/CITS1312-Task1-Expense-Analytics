## 📌 Project Scope

This project analyzes personal spending patterns over 12 months using interactive data visualizations. It helps track where money is being spent, compare budget vs actual expenses, and monitor savings growth over time.

---

## 📊 Features

- Monthly Expense Trend (Line Chart)
- Category-wise Breakdown (Doughnut Chart)
- Budget vs Actual Comparison (Bar Chart)
- Cumulative Savings Trend (Line Chart)
- Recent Transactions Table with categories

---

## 🔍 Key Insights

- **Total Annual Spending:** ₹42,850
- **Highest Category:** Food at 28.9%
- **Average Monthly Savings:** ₹3,571
- **Overspent Months:** October, January, March
